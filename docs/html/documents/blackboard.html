The Blackboard breakout & controller expansion, goes on top of your Elk Pi, providing an easy, solderless start to your Elk Audio OS project.

For complete examples on how to create musical devices with the Elk Pi and Blackboard, head to the examples section of this documentation.

Blackboard Page¶

For more information, and to purchase, head to the dedicated page on the elk.audio website.

Open-Source Hardware¶

The Blackboard, like the Elk Pi, is Open-Source hardware. For schematics and production files, please refer to the dedicated GitHub repository.
